# Focus DJ

Binaural beats plus brown and pink noise, a reactive visualizer, and an ADHD task
anchor so the timer never loses the one thing you sat down to do. Put on headphones.

**Runs fully offline in one tab: no account, no tracking, no data leaves your machine.**

## Use it
- Live: https://kalani.place/lab/focus-dj
- Or download `index.html` and open it in any browser.

## Tech
One self-contained HTML file. Vanilla JS + WebAudio. No build step, no dependencies.
